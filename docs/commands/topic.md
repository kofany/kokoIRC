# /topic

## Syntax

    /topic [channel] <text>

## Description

Set the topic for a channel. With just text, sets the topic for the
current channel. Specify a channel name to set a different channel's topic.

## Examples

    /topic Welcome to the channel!
    /topic #linux Welcome to #linux
